Tag Information
About Tags

Tags help categorize and organize feedback within departments. They allow students to label their feedback with specific topics and make it easier to identify trends and common issues.

Each tag is associated with a specific department and can only be used for feedback submitted to that department.

Best Practices
  • Clear Names - Use descriptive, concise names that clearly indicate what the tag represents.
  • Helpful Descriptions - Provide descriptions that explain when and how to use the tag.
  • Consistent Patterns - Use consistent naming patterns for related tags (e.g., "Course Content", "Course Delivery", etc.).
  • Balanced Coverage - Create enough tags to cover common feedback topics, but not so many that it becomes overwhelming.
Tip: Consider creating tags for common feedback categories such as "Facilities", "Course Content", "Teaching Style", "Technology Issues", "Scheduling", and "Academic Support".
